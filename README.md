Monolake: A light-weight high-performance Rust-based framework for customized proxy fast development  
====================================================================================================

[![Documentation Status](https://readthedocs.org/projects/monolake/badge/?version=latest)](https://monolake.readthedocs.io/en/latest/?badge=latest)

``Monolake`` is an **light-weight and high-performance** framework to **fast** develop customized proxy, it provides the basic common functionalities for proxy such as HTTP protocol process, connection pool management etc., while allows to implement customized handler. The framework is written in **Rust** that also inherits the safety, speed of the languages.
